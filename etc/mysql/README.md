# Que és el MYSQL

MariaDB és un sistema de gestió de bases de dades derivat de MySQL.

# Per que hem escollit aquest programa com base de dades?

Hem escollit aquest programa perquè té una facilitat a l'hora de crear base de dades i és molt intuïtiu.
